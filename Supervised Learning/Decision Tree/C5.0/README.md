# Algoritma C5.0

## Description
Algoritma C5.0 merupakan penyempurnaan dari algoritma C4.5 dan ID3. Dalam proses pembentukan pohon keputusan, nilai *information gain* yang tertinggi akan terpilih sebagai root bagi node selanjutnya.
